# 🎂 Juliana's 1st Birthday - Digital Invitation

A premium, interactive single-page scrollable experience for Juliana's 1st birthday celebration featuring parallax effects, smooth animations, and delightful interactions.

## ✨ Features

- **Envelope Opening Animation**: Click-to-reveal entrance with smooth transitions
- **Parallax Scrolling**: Multi-layered background with depth and motion
- **Floating Elements**: Confetti, paw prints, and balloons that drift as you scroll
- **Staggered Text Animations**: Letters bounce in like a playful kitten
- **Interactive RSVP Button**: Pulse animation with hover effects and sparkle feedback
- **Fully Responsive**: Optimized for mobile and desktop viewing

## 🎨 Design Palette

- **Soft Rose**: #FFC0CB
- **Champagne Cream**: #FFFDD0
- **Metallic Rose Gold**: #FFD700 accents
- **Typography**: Quicksand for headers, system sans-serif for body

## 🚀 Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📝 Customization

Edit the event details in `components/InvitationContent.tsx`:
- Date, time, and location
- Theme and messaging
- RSVP button behavior

## 🛠️ Tech Stack

- **Next.js 15** (App Router)
- **Framer Motion** for animations
- **Tailwind CSS** for styling
- **TypeScript** for type safety

## 📱 Mobile Optimization

The parallax effects are optimized for touch devices with fluid scaling and performance considerations.
